# Свободная онлайн библиотека
##Назначение: 

Хранение книг и обмен ими между пользователями.

###Роли:
* Гость
* Зарегистрированный пользователь

##Гость может просматривать краткую информацию о книгах на главное странице, скачивать их.

##Зарегистрированный пользователь может:
* Смотреть расширенную иинформацию о книгах.
* Оставлять комментарии
* Добавлять в библиотеку собственные книги, редактировать их и удалять.
